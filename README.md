# Antibuddy
Automates the data extraction process for the Doris Lab. 

## Required Tools
- samtools-1.9
- seqtk
- fatotwobit
- blat
- blatSrc

## Usage
1. Create a folder in the format bio-sample-[number].
2. Place the .bam file within this folder.
3. Have a location for strain specific target files.
